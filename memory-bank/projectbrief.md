# Maison Delice - Project Brief

## Overview
Maison Delice is a boutique French patisserie website for a fictional bakery located in Nisantasi, Istanbul. The site showcases artisan pastries, cakes, macarons, and breads with a warm luxury aesthetic.

## Target Audience
- High-end food enthusiasts in Istanbul
- People looking for custom cakes for special occasions
- French patisserie lovers

## Key Requirements
- 7-page website (Home, Menu, About, Gallery, Order, Blog, Contact)
- Warm luxury design: dark brown, cream, gold tones
- Script typography for branding, serif for headings, sans-serif for body
- Organic animations (0.6-0.8s transitions)
- Fully responsive design
- No build errors

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- Framer Motion
- GSAP (available but not primary)
- Lenis (smooth scroll - available)

## Brand Identity
- Name: Maison Delice
- Tagline: Artisan Patisserie - Istanbul
- Location: Nisantasi, Istanbul
- Founded: 2018
- Chef: Deniz Aydin (fictional)
