# Maison Delice - Product Context

## Why This Project Exists
This is a portfolio project demonstrating the ability to create a premium, visually rich website for a luxury food brand. It showcases advanced frontend techniques including animations, responsive design, and modern React patterns.

## Problems It Solves
- Provides an elegant online presence for a boutique patisserie
- Allows customers to browse the menu and view pricing
- Enables custom order requests through a comprehensive form
- Shares the brand story and builds trust through about/blog pages
- Provides gallery showcase of products and venue

## User Experience Goals
- Immediate sense of luxury and warmth upon landing
- Smooth, organic animations that enhance rather than distract
- Easy navigation between all 7 pages
- Mobile-first responsive design
- Fast loading with optimized assets
